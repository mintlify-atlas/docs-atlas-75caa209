## Overview

Eventos API is a REST API service built with Node.js and Express that provides access to event participant data. It connects to a PostgreSQL database and exposes endpoints for querying participants associated with events hosted at UTMACH.

| Configuration | dotenv 16.x |
| CORS | cors 2.8.x |

## How it works

The API starts an Express server that connects to a PostgreSQL database using a connection pool. When a request arrives at `/lista`, the server queries the `Prueba` table and returns all rows as a JSON array. The allowed frontend origin is `https://eventosutmach.000webhostapp.com`.

<Note>
  The API currently exposes a single endpoint. See the [API Reference](/api-reference/overview) for full details.
